IDM 241: Microinteractions - Beta

Pinterest saving pin - https://www.pinterest.com

Beta Descriptions

Triggers
  • Hover on save button
  • Hover off save button
  • Click on save button
  • Click off save button
Rules

Hover on button

  • Cursor changes to pointer
  • Color changes to #9E2224

Hover off button

  • Pointer changes to cursor off hover or zoom-in cursor (if still hover on image)
  • Color changes to original shade

Click on button

  • "Save" text changes to "Saved"
  • Color changes to black
  • Drop down menu left of button turns to link
  • Pop up message slide up, stays 3s, expands and fades out
  • "Save" record is set to true
  • Share and More button slides out to right
  • Edit button fades in

Click off button

  • "Saved" text changes to "Save"
  • Color changes to original shade
  • Link left of button turns to dropdown menu
  • Pop up message slide up, stays 3s, expands and fades out
  • "Save" record is set to false
  • Edit button fades out
  • Share and More button slides in from right

Click on button

  • Editing button appears to edit saved pin details
  • Pop up message appears

Click off button

  • Editing button disappears
  • Pop up message appears
